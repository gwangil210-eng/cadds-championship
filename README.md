# CADDS Championship

CADDS Championship is a contest-oriented workspace for rebuilding and presenting the CADDS proposal as a reviewable web, document, visual-reference, and prompt asset package.

## Project Purpose

This repository organizes CADDS contest materials into a structured submission package.  
The goal is not to claim a production-ready system, but to present a minimum viable contest package based on documentation, web assets, visual references, and prompt resources.

## Core Principle

CADDS follows a controlled AI decision-support model:

- AI generates drafts and assists with analysis.
- Human reviewers make the final judgment.
- Important decisions must remain auditable.
- Unsupported performance claims, fixed thresholds, and unverifiable numerical assertions are avoided.

## Repository Structure

| Folder | Purpose |
|---|---|
| `01_source` | Original source materials and reference inputs |
| `02_docs` | Submission documents, summaries, and review materials |
| `03_visual_refs/01_current_screenshots` | Current UI screenshots and visual reference assets |
| `04_prompt` | Prompt assets used for web, document, slide, and video generation |

## Current Scope

This repository is a contest submission workspace.  
It focuses on proposal clarity, evaluator readability, documentation structure, and asset management.

It is not yet a production software system.

## Review Notes

- Public template status is used for reviewability and reuse.
- This repository should be evaluated as a structured contest package.
- Technical claims should be interpreted conservatively unless supported by linked evidence or attached documents.
