# 캐즈 챔피언십

CADS 웹 재구축 및 전략 작업 공간

## 프로젝트 개요
이 저장소는 캐즈 챔피언십 관련 자료를 한곳에서 정리하고,
웹 재구축, 문서 작성, 시각 레퍼런스 수집, 프롬프트 자산 관리를 체계적으로 진행하기 위한 작업 공간입니다.

## 목적
- 웹 재구축 방향 정리
- 브랜드/콘텐츠 전략 문서화
- 시각 레퍼런스 아카이빙
- AI 프롬프트 및 제작 자산 관리
